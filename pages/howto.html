How to use the REST API

An user is defined by 5 fields:

  • _id (number)
  • name (string)
  • email (string)
  • city (string)
  • hasACat (boolean)

To use the api, you need to use the address http://localhost:3000/v1 and

/users: no parameter - lists all users

/users/{an _id number here}: the _id parameter - find an user baed on their ID
For example, test http://localhost:3000/v1/users/0

/addUser: takes the 5 fields as parameters and add the user in the JSON file (no GET, only POST)
For example, test http://localhost:3000/v1/addUser?_id=9&name=shigeru&city=tokyo&email=shigeru@mycompany.co&hasACat=true on a POST application (like Postman)

/deleteUser: takes the user id as a parameter and deletes the user from the json file
For example, test http://localhost:3000/v1/deleteUser?_id=1
